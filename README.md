## Run Locally

Clone the project

```bash
  git clone https://github.com/yarpo/wolo-app.git
```

Go to the project directory

```bash
  cd wolo-app
```

Install dependencies

```bash
  npm install
```

Start the application

```bash
  npm start
```

## Running Tests

To run tests, run the following command

```bash
  npm test
```
